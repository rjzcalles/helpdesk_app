# helpdesk_app

version 1.0.0

## Installation

1. Clone the repository

on the server folder client run this command:
1. Run `npm install`
2. Run `npm start`

on the server folder server run this command:
1. Modify the .env file
2. Run `npm install`
3. Run `npx sequelize-cli db:migrate`
4. Run `node index.js`
